# Linux-Basic

## 1. 常用命令

### 1.1 常用命令

ls 列出文件，及文件夹

ls -l 列出文件夹详情

ls -a 列出文件夹的所有内容，包括以`.`

cat 读取指定文件的内容并打印到终端数据

touch a.txt 创建文件

touch a.txt b.txt 

touch file_{1..10}.txt 创建10个文件

### 1.1 用户相关

sudo adduser--添加用户

su root 切换用户

groups root 查看当前用户属于哪一个用户组

who am i当前伪登录用户

whoami当前用户

sudo deluser lilei --remove-home删除用户
